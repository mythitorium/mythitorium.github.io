dl-bfg.exe

dl-bfg 'batch file generator' is a simple program which, upon you copying something to your clipboard, will save that content to a text file.

This was created to help make url batch files for youtube-dl. Instead of putting a copied url into a text file yourself, dl-bfg does it for you. You also get to exclude saving clipboard contents if they do not contain a given string.

Really that's all there is to it, not anything complex.
